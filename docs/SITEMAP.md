# Sitemap — NusaNexus Router MVP

## Public
- `/` — landing page
- `/login` — login
- `/signup` — signup

## Private / Onboarding
- `/onboarding` — first-time setup wizard

## Dashboard
- `/dashboard` — home overview
- `/dashboard/providers` — provider connections list
- `/dashboard/providers/:id` — provider detail
- `/dashboard/presets` — routing presets list
- `/dashboard/presets/:id` — routing preset detail
- `/dashboard/endpoint` — endpoint, API key, config snippets
- `/dashboard/usage` — usage and monitoring
- `/dashboard/settings` — workspace and security settings

## Supporting UI States
- provider disconnected
- provider expired
- quota exhausted
- fallback exhausted
- invalid API key
- no workspace found
- no provider connected
